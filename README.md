# Nomie

Nomie is an iOS app project built with SwiftUI.

## Requirements

- Xcode (latest stable recommended)
- iOS Simulator or a physical iPhone

## Getting started

1. Open `Nomie.xcodeproj` in Xcode.
2. Select a target device or simulator.
3. Press Run.

## Project structure

- `Nomie/` — App source code
- `NomieTests/` — Unit tests
- `NomieUITests/` — UI tests
